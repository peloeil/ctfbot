# times チャンネル (times)

## 概要

`/times create` コマンドで `times` カテゴリ配下にテキストチャンネルを一括作成する。

## コマンド

### `/times create names`

`names` はカンマ（`,`、`、`）または改行で区切って複数指定可。

処理:
1. 各名前を正規化（lowercase、`[^a-z0-9\-]` を `-` に、連続 `-` を 1 つに、先頭末尾 `-` 除去）
2. guild のカテゴリから名前が `times` のものを検索。なければ「times カテゴリが見つかりません。」
3. カテゴリ内に同名チャンネルが既存 → スキップ
4. なければ作成

応答:
- 作成あり: 「✅ #name1, #name2 を作成しました。」
- スキップあり: 「⏭️ #name3 は既に存在します。」
- 両方あれば両方表示
